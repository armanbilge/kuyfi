Scala Java-Time

This project provides a parser for the tzdb data files and can generate source code that can be imported by some other projects

Usage

The kuyfi library is currently available for Scala (JVM, version 8 and later) Both Scala 2.10 and Scala 2.11

To get started with SBT, add one (or both) of these dependencies:

  • `libraryDependencies += “io.github.cquiroz” %% “kuyfi” % “0.6.0”

Building

This project builds using sbt.

Testing

Tests are executed via

sbt test

Note that the tests compare the output with the timezone support on the JVM hence they may not match to any JVM version At the moment the test use the JDK 1.8 b31 to match the version present in travis

Bug

Note that there are some border cases errors with the latest versions of the jdk, specifically with 1.8 b144
